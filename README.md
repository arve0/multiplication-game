# HTML5 multiplication game

A simple game made with [React] and [redux]. [Play it!]

![](demo.gif)

## Develop

```bash
git clone https://github.com/arve0/multiplication-game
cd multiplication-game
npm install
npm start
```

## Dist
```bash
npm run build
```
Distribute files in [dist](dist).
